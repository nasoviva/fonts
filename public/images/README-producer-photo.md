# Портрет продюсера

Источник: [Instagram DD_ALSdOVAc](https://www.instagram.com/p/DD_ALSdOVAc/?img_index=2) (второе фото в карусели, `img_index=2`).

1. Откройте ссылку в браузере (войдите в Instagram при необходимости).
2. Выберите нужный кадр (img_index=2).
3. Сохраните изображение как **`zhanna-kolesnik.jpg`** в эту папку (`public/images/`).
4. Перезапустите `npm run dev:clean` и обновите сайт (**Cmd+Shift+R**).

Автозагрузка (если сеть доступна): `npm run fetch:producer`
